Zod Request is a library for validating [HTTP Request](https://developer.mozilla.org/en-US/docs/Web/API/Request) using [Zod](https://zod.dev).

## Prerequisites

- Node.js 18+
- Zod 4

## Installation

```package-install
npm install @nicnocquee/zod-request
```
